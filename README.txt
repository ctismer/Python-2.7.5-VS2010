This project is a copy of the Python 2.7.5 source code downloaded from

http://www.python.org/ftp/python/2.7.5/Python-2.7.5.tar.bz2

including some modifications documented at

http://wiki.python.org/moin/VS2010

to enable compilation with Visual Studio 2010.
